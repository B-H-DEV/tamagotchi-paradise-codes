# Tamagotchi Paradise — Ultimate Codes Checklist

This repository contains a consolidated list of **all publicly known Lab Sticker codes and Shop codes**
for Tamagotchi Paradise (Food, Snacks, Accessories, Outside Toys, Deco Items, Plushies, etc.) as of July 12, 2026.

## Files

- `tamagotchi-paradise-codes-checklist.md`  
  Markdown checklist with checkboxes, grouped by category.

- `tamagotchi-paradise-codes-onepage-3col-bigfont.pdf`  
  Printable one-page A4 PDF (3 columns, larger font) for physically ticking off codes.

## Sources / Cross-check

Aggregated and cross-checked against:
- Tamagotchi Paradise/List of Codes on Fandom.
- JP攻略wiki / Ameblo shop & lab code lists.
- Thai guide, Tamavault, Tamagotchi Center.
- Reddit “Item Code Checklist” and “COMPLETE SHOP AND LAB CODES FOR TAMAGOTCHI PARADISE!!!”.

If new promo codes appear (collabs, events, DLC), they can be added to the Markdown and PDF in future revisions.
